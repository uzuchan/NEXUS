## ゴール

採用担当者がスマホでURLを開いても美しく・速く動くNEXUSにする。

### Definition of Done

1. **モバイル性能**: iPhone縦相当（390×844, DPR3エミュレーション）で平均45fps以上、UI崩れ・横スクロールなし
2. **アダプティブ品質**: FPS計測で high / mid / low の3段階を自動切替（粒子数・bloom解像度・DPRを段階制御）
3. **reduced-motion対応**: `prefers-reduced-motion` でカメラ演出・大型アニメーションを静的フォールバックに
4. **OGPリンクカード**: メール・チャットにURLを貼るとヒーロー画像つきカードが出る（`public/og.jpg` 1200×630 + meta）
5. **QA合格**: 契約違反ゼロ・consoleクリーン・headless Chromeモバイルエミュレーション実機検証 → 再撮影 → push公開

## 品質API契約（v1.1追加、全エージェント厳守）

- `SceneManager` が rolling FPS（約2秒窓、ヒステリシス3秒）で tier を決定: `<28 → 'low'`, `<45 → 'mid'`, それ以外 `'high'`
- `sceneManager.quality` で現在tierを公開。register済みオブジェクトに `setQuality(tier)` があれば tier変化時に呼ぶ（任意実装・後方互換）
- DPR: high=`min(devicePixelRatio,2)` / mid=1.5 / low=1（SceneManagerが所有）
